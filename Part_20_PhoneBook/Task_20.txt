Разработать программу "Телефонная книга", которая позволяет хранить следующую информацию о контактах человека:
• номер
• ФИО (имя)
• дата рождения
• адрес
• телефон
• текстовое описание контакта

Программа должна использовать динамический массив структур для хранения телефонной книги. Программа имеет меню, в котором есть следующие функции:
• просмотр всех контактов
• добавить контакт
• удалить контакт
• изменить контакт (поменять телефон, адрес, описание, дату рождения)
• поиск по имени
• поиск по телефону
• загрузить / сохранить телефонную книгу в файл
• выход
